# 📁 Folder Structure

```
root
 ├─ packages/
 │   ├─ shared/
 │   ├─ utils/
 │   ├─ config/
 ├─ apps/
 │   ├─ web/
 │   └─ native/
 ├─ projects/
```

## Explanation

- `packages/`: shared logic, components, configurations
- `apps/`: main applications by platform
- `projects/`: legacy or special grouping
- `.husky`, `.turbo`: Git hooks & Turborepo config
